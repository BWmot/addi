# 加密导出/导入

> 更新时间：2026-04-07

---

## 加密算法

| 参数     | 值                  |
| -------- | ------------------- |
| 算法     | AES-256-GCM         |
| 密钥派生 | PBKDF2 (100,000 次) |
| 哈希     | SHA-512             |
| Salt     | 16 字节（随机）     |
| IV       | 12 字节（随机）     |
| Tag      | 16 字节             |

---

## 数据结构

### 加密配置格式

```json
{
  "version": 1,
  "exportedAt": 1699900000000,
  "encryptionApiKey": "base64(salt + iv + ciphertext + tag)",
  "providers": [...]
}
```

### 加密数据布局

```
salt (16) + iv (12) + ciphertext (可变) + tag (16)
```

---

## 流程

### 导出

```
用户输入密码 → 加密 ApiKey → 保存加密数据
用户留空 → 不导出 ApiKey
```

### 导入

```
检测 encryptionApiKey → 提示输入密码 → 解密 → 验证 tag → 成功/失败
```

---

## 错误处理

| 场景                  | 处理                  |
| --------------------- | --------------------- |
| 密码错误              | 导入配置但跳过 ApiKey |
| 数据被篡改 (tag 失败) | 同上                  |
| 无加密字段            | 正常导入              |

---

## 安全措施

- AES-256-GCM 提供加密 + 完整性验证
- PBKDF2 防止暴力破解
- 随机 salt/IV 防止已知明文攻击
